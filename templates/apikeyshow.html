Что такое API-ключ?
API ключ (Application Programming Interface ключ) - это уникальный идентификатор, который используется для авторизации запросов к API сервису. Он позволяет разработчикам и пользователям доступиться к функциональности API, используя определенные методы и параметры.

Для чего нужен наш API-сервис?

API сервиса Edy.boost предоставляет возможность учителям автоматизировать процессы работы с группами учеников, выдачи тестов и импорта тестов в удобные для работы форматы (.json, .docx)

Для кого доступен API?

На данный момент API доступен для пользователей, указавших при регистрации роль "Учитель". Для таких пользователей API доступен в рамках их доступа (т. е. в рамках групп, которые они создали)
